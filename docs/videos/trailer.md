# Asylum cinematic trailer

**Target duration:** 75–90 seconds  
**Format:** 1920×1080, 16:9, H.264/AAC  
**Voice:** cinematic English narration  
**Purpose:** introduce the problem, reveal the fleet model, and end on the core
promise: build in parallel, decide with evidence, merge the winner.

## Story

The trailer begins in isolation: one difficult task and one serial attempt. The
visual system fractures that line into several independent routes, revealing
Asylum as the coordinating environment. The middle accelerates through
worktrees, live activity, checks, diffs, and annotations. The final movement
converges on a selected run and the Asylum title.

## Narration

1. Every difficult task begins the same way. One question. Too many possible
   answers.
2. One agent. One attempt. One path through the unknown.
3. Asylum turns a single prompt into a fleet.
4. Each agent gets its own branch. Its own worktree. Its own chance to win.
5. See who is working. Who is blocked. Who is done. And what every run changed.
6. Run checks. Read the diff. Annotate the exact line. Send it back.
7. Not the loudest answer. Not the first answer. The answer that survives the
   evidence.
8. Build in parallel. Decide with confidence. Merge the winner.

## Visual rules

- Cyan represents dispatch and intelligence.
- Amber represents verification and risk.
- Mint represents convergence and success.
- Coral appears only for failure and interruption.
- Movement follows routes and state changes; it is never decorative.
- No protected franchise imagery, logos, characters, music, or footage.
